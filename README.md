# chuck design — chuckdesign.com

Next.js 16 App Router marketing site for chuck design, North Port FL.

## Stack

- Next.js 16 App Router · JavaScript
- Tailwind v4 via `@import "tailwindcss"` in `app/globals.css`
- Phosphor Icons (CDN) · Outfit + Geist Mono (next/font/google)
- Formspree for contact form · Vercel for hosting

## Dev

```bash
npm run dev   # localhost:3000
npm run build
vercel        # production deploy
```

## Routes

| Route | Description |
|---|---|
| `/` | Main marketing site |
| `/portfolio` | 20-project carousel |
| `/sitemap.xml` | Auto-generated by `app/sitemap.js` |
| `/robots.txt` | `public/robots.txt` |

## Before go-live checklist

### 🔴 Still needed

- [ ] **`public/images/apple-touch-icon.png`** — 180×180 PNG from the favicon coral-dot artwork. Currently referenced in `app/layout.js` but the file doesn't exist. iOS home screen icon silently fails without it.

- [ ] **Google Analytics** — No tracking is installed. Add a GA4 measurement script to `app/layout.js`, or enable Vercel Analytics in the Vercel dashboard (one click, no code needed).

- [ ] **Privacy policy page** — The contact form collects name, email, phone, and message. Add an `app/privacy/page.js` and link it in the footer. Builds trust and covers you legally.

- [ ] **Confirm About portrait** — `components/About.jsx` uses `public/images/family.png`. Make sure that's the photo you want clients to see.

- [ ] **Social links in footer** — `components/Footer.jsx` has no social media links. Add Instagram, Facebook, or LinkedIn if chuck design has active profiles.

- [ ] **Pricing signal** — All CTAs go straight to "Get a free quote" with no price anchoring. A "Starting at $X" line somewhere filters bad-fit leads before they reach you.
